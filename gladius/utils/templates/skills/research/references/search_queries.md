## Kaggle Discussion Searches

```
# Competition-specific
site:kaggle.com/competitions/{COMPETITION_ID}/discussion
site:kaggle.com/competitions/{COMPETITION_ID}/discussion data leak
site:kaggle.com/competitions/{COMPETITION_ID}/discussion feature engineering
site:kaggle.com/competitions/{COMPETITION_ID}/discussion 1st place solution

# Past similar competitions
site:kaggle.com {TASK_TYPE} competition winner write-up solution summary 2024
```

**What to prioritise in discussions:**
1. Data leaks or shortcut features (highest impact, easy to implement)
2. External datasets that are allowed
3. Winning solution write-ups (post-competition)
4. Novel feature engineering specific to this domain
